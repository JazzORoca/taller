# Front-End Blog

Falta conectar a la API

This is a blog built with [Nuxt.js](https://nuxtjs.org/) and [Vue.js](https://vuejs.org/).

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install
```

## Development

Start the development server on http://localhost:3000

```bash
#yarn
yarn dev

#npm
npm run dev
```
